# Digizone Solutions -- Professional Portfolio Website

## Project Overview

Digizone Solutions is a modern **single‑page company portfolio website**
developed as part of the **Advanced Web Technologies** course
assignment.\
The project demonstrates responsive web design, interactive UI
components, and modern frontend development practices using HTML, CSS,
Bootstrap, and JavaScript.

## Technologies Used

-   **HTML5** -- Semantic structure and layout\
-   **CSS3** -- Styling, animations, variables, and visual effects\

## Website Sections

The website is built as a **single-page application structure**
containing the following sections:

1.  **Header & Navigation**
    -   Fixed glassmorphism navbar
    -   Navigation links
    -   Dark/Light mode toggle
2.  **Hero Section**
    -   Animated gradient background
    -   Introduction to Digizone Solutions
    -   Call-to-action button
3.  **Academic Information**
    -   Displays assignment details including course, instructor,
        student name, and registration number.
4.  **About Me**
    -   Student profile section
    -   Professional image with CSS pulse animation
    -   Academic and technical background
5.  **Services Portfolio**
    -   Web Development
    -   Flutter/Dart Software Development
    -   UI/UX Design
    -   Cloud Hosting
6.  **Contact Section**
    -   Company contact details
    -   Contact form with validation

## Key Features

### Dark Mode / Light Mode

-   Implemented using **JavaScript and CSS variables**
-   Theme preference stored in **localStorage**

### Dynamic Service Filtering

-   Users can filter services by categories:
    -   All
    -   Development
    -   Design
    -   Cloud & Infrastructure

### Form Validation

-   Email validation using **Regular Expressions (Regex)**
-   Uses Bootstrap validation classes for user feedback

### Scroll Reveal Animations

-   Implemented using **IntersectionObserver API**
-   Smooth fade-in animations when elements enter the viewport

### Advanced UI Enhancements

-   Glassmorphism navigation bar
-   Animated gradient hero background
-   Hover micro‑interactions
-   Custom styled scrollbar
